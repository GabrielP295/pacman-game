# Pac-man

A collaborative recreation of the classic game, Pac-man

See this [Design Doc](https://docs.google.com/document/d/16V4dqQ7mHpZwIlFFpnZazDmEYS1H-7WXvRmzPMPT5Es/edit?tab=t.0) for more details about the design.

### Requirements

* Any modern browser

### How to run

* Go to [https://gabrielp295.github.io/pacman-game/](https://gabrielp295.github.io/pacman-game/)

* Use keyboard arrow keys to control
